# UTS_DesainWeb_Teori
Disini saya membuat sebuah website sederhana yang digunakan untuk para penjoki tugas untuk membantu mempermudah ketika ada customer yang ingin memesan jasa joki tugas tersebut dan disini para mahasiswa / pelajar yang malas dapat memesan jasa joki tersebut yang ada dalam website ini bila ingin pesan maka akan terhubung ke whatsapp pribadi admin tersebut untuk membantu tugas apa yang ingin dikerjakan disini terdapat 4 halaman dimana beranda itu menggunakan file index.html, pesan joki menggunankan joki.html, testimoni menggunakan testimoni.html, about us menggunakan about.html, disini juga terdapat file css dan js nya untuk merapikan isi dari website tersebut dan menambah responsive / lebih interaktif terhadap user nya.
